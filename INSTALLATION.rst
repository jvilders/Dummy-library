============
Installation
============

At the command line::

    $ pip install dummylibrarylib

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv dummylibrarylib
    $ pip install dummylibrarylib

Or, if you are using pipenv::

    $ pipenv install dummylibrarylib

Or, if you are using pipx::

    $ pipx install dummylibrarylib
